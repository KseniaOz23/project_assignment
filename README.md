# Задание 1.1: Разработка ETL-процесса загрузки данных
## Описание
ETL-процесс загружает данные из CSV файлов в PostgreSQL. Реализован на Apache Airflow.
## Файлы
- `bank_etl_dag.py` - основной DAG
- `sql/` - SQL скрипты для вставки/обновления данных
- `create_tables.sql` - создание схем и таблиц
## Функциональность
1. Загрузка справочников (MD_ACCOUNT_D, MD_CURRENCY_D и др.)
2. Загрузка фактов (FT_BALANCE_F, FT_POSTING_F)
3. Upsert (вставка или обновление) при конфликтах
4. Логирование в таблицу LOGS.ETL_LOG
